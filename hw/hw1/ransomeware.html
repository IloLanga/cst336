Definition

A Computer Worm is a type of malicious software that threatens to publish the victim's data or perpetually block access to it unless a ransom is paid. Ransomware attacks are typically carried out using a Trojan that is disguised as a legitimate file that the user is tricked into downloading or opening when it arrives as an email attachment.


The WannaCry ransomware attack started the 12 May of 2017 and ended the 15 May 2017, it targeted computers running the Microsoft Windows operating system by encrypting data and demanding ransom payment in Bitcoin.
